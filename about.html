Mapping LA County

Map My Website

  Mapping LA County uses data collected from government sources and presents those data in a layered format.

  The goal of this project was to enable a user to produce maps through which they can view different facets of the county of Los Angeles in different contexts.

  This project was undertaken in around 2016, and represents some of my first explorations into data visualization. All data presented in Mapping LA County have been gathered from data.lacounty.gov.

Code for this project can be accessed through GitLab.
