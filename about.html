Mapping LA County

Map My Website

  Mapping LA County uses data collected from government sources and presents those data in a layered format. The goal of this project is to enable the user to produce maps through which they can view different facets of the county of Los Angeles in different contexts. This is project is a work in progress, and I hope to find and incorporate more interesting data in the near future. All data in the current incarnation of Mapping LA County have been gathered from data.lacounty.gov.

Code for this project can be accessed through GitLab.
